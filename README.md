# Network Traffic Classification Using Transformer-Enhanced LSTM

## Description
This project classifies network traffic using a deep learning model combining LSTM and Transformer attention mechanisms.

## Datasets
- Dataset 1: https://www.kaggle.com/datasets/nameera06/network-traffic
- Dataset 2: https://www.kaggle.com/datasets/dhoogla/cicdarknet2020

## Model Architecture
- LSTM layer (64 units)
- Multi-Head Attention (4 heads)
- Layer Normalization
- Global Average Pooling
- Dropout (0.3)
- Dense output layer
